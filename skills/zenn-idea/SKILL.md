# Zenn Idea 記事スキル

会話コンテキストからZennのアイデア記事を生成し、Markdownファイルとして保存する。

読み物として膨らませなくて良い。共感も詰めなくて良い。

## 記事の構成

フロントマター
アイデア
使えそうなサービスや参考になる情報
コード片（なければ省略）
なぜその設計か

## ファイル保存

カレントディレクトリの `.zenn-idea-config` を読み込む。なければ保存先ディレクトリを聞いて作成する。

```
output_dir: ./articles
```

ファイル名案（`idea-<theme>.md`）とともにプレビューをチャットに出力し、承認後に保存する。
