# code-generator スキル

指定された言語と目的に基づいて、関数コードを一つ生成して返す。

## 入力

呼び出し時には以下の情報を受け取る（プロンプトに含まれる）：

- **言語** — 生成する言語（例：Python、TypeScript、Go、Rust、Ruby、Java など）
- **目的** — 何をする関数かの説明

どちらかが欠けている場合は、コードを生成する前にユーザーに確認する。

## 生成ルール

1. 一回の実行で**関数を一つだけ**生成する
2. 関数名・引数名・変数名は**英語**で書く
3. 関数の先頭に**短いドキュメントコメント**を付ける（その言語の標準スタイルで）
4. 実用的なコードを書く — エラーハンドリングは必要な範囲で入れるが、過剰にしない
5. コードブロック（` ``` ` 付き）で返す

## 出力形式

生成したコードのみを返す。余計な説明は不要。

**例：**

呼び出し：「Pythonでリストの重複を除くユニーク化関数を作って」

出力：
```python
def unique(items: list) -> list:
    """Return a list with duplicates removed, preserving order."""
    seen = set()
    result = []
    for item in items:
        if item not in seen:
            seen.add(item)
            result.append(item)
    return result
```

## 言語が指定されていない場合

文脈から推測できる場合は推測して生成する。推測できない場合は「どの言語で生成しますか？」と一言聞く。
